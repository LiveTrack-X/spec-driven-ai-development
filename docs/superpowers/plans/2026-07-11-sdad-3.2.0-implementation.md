# SDAD 3.2.0 Packet Consistency Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement the approved state-v2 packet-consistency contracts without changing existing v1 behavior, prepare a fully verified local v3.2.0 release candidate, and keep push, tag, and publication behind explicit owner gates.

**Architecture:** The existing parser, five-check Doctor pipeline, confined project view, and checkout-only CLI remain. State version dispatch is added to the shared state contract; v2 validation, continuity, and ledger rules branch inside the existing checks so v1 check names and report behavior stay exact. One canonical agent kernel and one canonical no-clone prompt continue to generate or synchronize all public copies.

**Tech Stack:** Python 3.10+ standard library, `unittest`, Markdown, JSON, PowerShell, Bash, Git, GitHub CLI.

class DiagnosticError(RuntimeError):

        self,

def _probe_version_requirement(
    arguments: Sequence[str],
) -> _VersionRequirementProbe:
    if not arguments or arguments[0] != "doctor":
        return _VersionRequirementProbe(False, None, None)
    values: list[str] = []
    index = 1
    while index < len(arguments):
        token = arguments[index]
        if token == "--":
            break
        if token == "--require-version":
            if index + 1 >= len(arguments) or arguments[index + 1].startswith("-"):
                return _VersionRequirementProbe(True, None, "--require-version requires one value")
            values.append(arguments[index + 1])
            index += 2

        if token.startswith("--require-version="):

        index += 1
    if not values:

    if len(values) != 1:
        return _VersionRequirementProbe(True, None, "--require-version may appear once")
    if _CORE_VERSION.fullmatch(values[0]) is None:
        return _VersionRequirementProbe(True, values[0], "--require-version requires core X.Y.Z")
    return _VersionRequirementProbe(True, values[0], None)
